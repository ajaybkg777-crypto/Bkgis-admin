# Admin Frontend

This is the admin panel for BKG International School.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

3. The admin panel will be available at `http://localhost:3000` (or another port if 3000 is taken).

## Routes

- `/login` - Admin login page
- `/panel` - Admin panel (protected)
- `/upload` - Upload Excel page (protected)

## Deployment

Deploy this separately from the main frontend. Ensure the backend API is accessible.
